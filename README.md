## プロジェクトの概要
プロジェクト名　Openlive
プロジェクトの概要　大学の軽音学部などに所属する学生が行うフェスのウェブページ。
## 始める方法

ブラウザで
https://openlive2.vercel.app/
を開いて結果を確認します。

app/page.tsx を変更してページの編集を開始できます。ファイルを編集すると、ページが自動的に更新されます。
